# Proyecto de hotel-front

Este proyecto fue desarrollado en React.

## Versión de node js

Node 16.14.2


## Instalación

1. Descarga el repositorio
2. Entrar a la carpeta desde la terminal 
3. Ejecutar `npm install` o `yarn`
4. Ejecutar `npm start` o `yarn start`
